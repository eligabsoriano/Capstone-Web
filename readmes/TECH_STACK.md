# Tech Stack

This file contains the technology overview for `Capstone-Web`.

## Core
- React 19
- TypeScript 5
- Vite (Rolldown)
- React Router

## UI
- Tailwind CSS 4
- shadcn/ui (Radix-based components)
- Lucide React
- Framer Motion

## Data and State
- TanStack Query
- Zustand
- Axios

## Forms and Validation
- React Hook Form
- Zod

## Testing
- Vitest
- Testing Library
- Playwright

## Code Quality
- Biome (lint + format)
- Husky
- lint-staged

## Utility Libraries
- Sonner
- Recharts
- TanStack Table
- date-fns
- react-error-boundary
